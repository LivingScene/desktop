# LivingScene

Presentations that stay alive — describe what you want, watch slides take shape, and present decks that keep themselves current.

This repository hosts the desktop app downloads. **[Get the latest release →](https://github.com/LivingScene/desktop/releases/latest)**

## Install

- **macOS (Apple silicon)** — download the `.dmg`, open it, and drag LivingScene to Applications.
- **Windows (x64)** — download the `Setup .exe` and run it.
- **Linux (x64)** — download the `.AppImage`, make it executable (`chmod +x`), and run it.

## Early-access builds

These builds aren't code-signed yet, so your OS will warn on first launch:

- **macOS:** right-click the app and choose **Open**. On newer macOS you may also need System Settings → **Privacy & Security** → **Open Anyway**.
- **Windows:** if SmartScreen appears, click **More info** → **Run anyway**.

New versions land on this page.
